# Product SPA

Одностраничное приложение на React + TypeScript с Zustand для управления каталогом продуктов. Данные загружаются с публичного API [Fake Store API](https://fakestoreapi.com/) и могут дополняться пользователем. Реализованы страницы каталога, карточки продукта и форма создания.

## Основные возможности

- Просмотр списка продуктов (`/products`) с фильтрацией, поиском без кнопки отправки и пагинацией
- Управление избранным с помощью лайков, удаление карточек, переход к деталям по клику
- Детальная страница (`/products/:id`) с расширенной информацией, лайком, удалением и редактированием
- Создание нового продукта на странице `/create-product` с валидацией через Zod и хранением в общем store
- Все данные (загруженные и созданные пользователем) сохраняются в Zustand store, лайки и удаление работают локально

## Стек

- React 18, TypeScript, Vite
- React Router 6 для маршрутизации
- Zustand для глобального состояния
- React Hook Form + Zod для работы с формами и валидацией
- Axios для работы с API

## Запуск

```bash
npm install
npm run dev
```

Приложение доступно по адресу `http://localhost:5173`.

## Деплой

Проект настроен для деплоя на Vercel или Netlify. Выберите один из вариантов:

### Вариант 1: Деплой на Vercel (рекомендуется)

1. **Установите Vercel CLI** (если еще не установлен):
   ```bash
   npm i -g vercel
   ```

2. **Войдите в Vercel**:
   ```bash
   vercel login
   ```

3. **Задеплойте проект**:
   ```bash
   vercel
   ```
   
   Следуйте инструкциям в терминале. Vercel автоматически определит настройки из `vercel.json`.

4. **Для продакшн деплоя**:
   ```bash
   vercel --prod
   ```

**Альтернатива через веб-интерфейс:**
- Зайдите на [vercel.com](https://vercel.com)
- Подключите ваш GitHub репозиторий
- Vercel автоматически определит настройки и задеплоит проект

### Вариант 2: Деплой на Netlify

1. **Установите Netlify CLI**:
   ```bash
   npm i -g netlify-cli
   ```

2. **Войдите в Netlify**:
   ```bash
   netlify login
   ```

3. **Задеплойте проект**:
   ```bash
   netlify deploy
   ```
   
   Для продакшн деплоя:
   ```bash
   netlify deploy --prod
   ```

**Альтернатива через веб-интерфейс:**
- Зайдите на [netlify.com](https://netlify.com)
- Перетащите папку `dist` (после `npm run build`) в область деплоя
- Или подключите GitHub репозиторий для автоматического деплоя

### Локальная сборка

Перед деплоем можно проверить сборку локально:

```bash
npm run build
npm run preview
```

Собранное приложение будет в папке `dist`.
